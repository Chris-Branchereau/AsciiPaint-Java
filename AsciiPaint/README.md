# AsciiPaint

This is a Java application that draws shapes in the command line console using ASCII characters. (2024 Exam Project)

## Overview
- **Type:** School / Personal Project
- **Language(s):** Java (Maven)
- **Author:** Chris Branchereau

## Features
- MVC design pattern
- Command line input processing

## Installation

```bash
git clone https://github.com/Chris-Branchereau/AsciiPaint-Java.git
cd AsciiPaint-Java/AsciiPaint

# Compile the project
mvn clean compile

# Package the application into an executable JAR
mvn clean package

# Run the executable
java -jar target/*.jar
